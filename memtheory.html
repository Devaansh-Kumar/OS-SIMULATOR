MFT Memory Management Technique

MFT which stands for "multiprogramming with fixed number of tasks" is a memory management technique in which memory is partitioned into fixed size and each job is assigned to a partition.

MVT Memory Management Technique

MVT which stands for "multiprogramming with a variable number of tasks" is a memory management technique in which each job gets just the amount of memory it needs, i.e. the partition is dynamic and changes as jobs enter and leave the system.
